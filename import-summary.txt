Moved Files:
------------
Android Gradle projects use a different directory structure than ADT
Eclipse projects. Here's how the projects were restructured:

* AndroidManifest.xml => simpleKML/src/main/AndroidManifest.xml
* assets/ => simpleKML/src/main/assets/
* libs/simple-xml-2.6.3.jar => simpleKML/libs/simple-xml-2.6.3.jar
* res/ => simpleKML/src/main/res/
* src/ => simpleKML/src/main/java/
